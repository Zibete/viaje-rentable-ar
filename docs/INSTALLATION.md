# Instalación de Viaje Rentable AR

Viaje Rentable AR está en etapa **alpha experimental**. Esta primera versión pública se distribuye como APK para instalación manual y puede tener errores de OCR o comportamiento incompleto en algunos dispositivos.

La app es solo un asistente visual e informativo: **no acepta ni rechaza viajes automáticamente**.

## Descargar el APK

1. Abrí la página de releases:
   <https://github.com/Zibete/viaje-rentable-ar/releases/latest>
2. Buscá la sección **Assets**.
3. Descargá el archivo:
   `viaje-rentable-ar-v0.1.0-alpha.apk`

## Instalar en Android

1. Abrí el APK descargado desde el teléfono.
2. Si Android lo pide, habilitá la instalación desde fuentes desconocidas para el navegador o administrador de archivos que estés usando.
3. Confirmá la instalación.
4. Abrí **Viaje Rentable AR**.

Android puede mostrar advertencias porque la app no está publicada en Google Play y se instala manualmente fuera de Play Store.

## Permisos necesarios

Para usar la app vas a necesitar otorgar permisos cuando Android los solicite:

- **Ventana flotante / Mostrar sobre otras apps**: permite mostrar la recomendación visual sobre la pantalla.
- **Captura de pantalla**: se solicita cuando se usa el monitoreo y permite leer información visible con OCR.

La captura debe ser autorizada por el usuario. La app no guarda capturas ni imágenes.

## Qué esperar de esta alpha

- Es una versión experimental para testing público/manual.
- Puede leer mal algunos textos por errores de OCR.
- Puede mostrar **DATOS INCOMPLETOS** si no detecta toda la información necesaria.
- No usa APIs privadas de apps de viaje.
- No toca botones ni manipula otras aplicaciones.
- No acepta ni rechaza viajes automáticamente.
